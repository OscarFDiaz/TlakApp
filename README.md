# 👨‍🎓🌐📚 Tlak!

📰 Tlak! es una aplicación para la memorización de palabras en el idioma inglés, creada usando <b>Vite + React, nextUI, tabler-icons, Firebase, Zustand, TailwindCSS, RouterDOM, TypeScript</b>.
